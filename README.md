# AI Business Suite

## Overview

AI Business Suite is a Streamlit-based application that provides multiple AI-powered business tools in one platform.

## Features

* Resume Builder
* Invoice Generator
* Caption Generator
* Spam Detector
* AI Support Chatbot

## Technologies

* Python
* Streamlit
* Scikit-learn
* Pandas

## Installation

pip install -r requirements.txt

python -m streamlit run app.py

* PDF Export
